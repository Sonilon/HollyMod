# 🛡️ ChatGuard

**Умный помощник модератора для Minecraft**

![Minecraft](https://img.shields.io/badge/Minecraft-1.20.1-brightgreen?style=for-the-badge&logo=minecraft)
![Fabric](https://img.shields.io/badge/Fabric-0.92.9-blue?style=for-the-badge)
![Java](https://img.shields.io/badge/Java-17+-orange?style=for-the-badge&logo=java)

*Автоматически следит за чатом, находит нарушителей и помогает быстро выдать мут — одним кликом.*

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 📖 **Чтение лога** | Читает `latest.log` напрямую — работает с любым форматом чата и Unicode-символами |
| ⚡ **Категории триггеров** | Слова разбиты по категориям с приоритетом, временем мута и пунктом правил |
| 🔔 **Красивые уведомления** | Оформленное сообщение в чате с кнопками **[ВСТАВИТЬ]** и **[КОПИРОВАТЬ]** |
| 🖥️ **HUD-оверлей** | Всплывающее уведомление в углу экрана с именем нарушителя |
| ✈️ **Telegram-логирование** | Каждое нарушение отправляется в ваш Telegram-канал/чат |
| ⚙️ **GUI настроек** | Полный редактор триггеров прямо в игре — нажми **[M / ь]** |
| 🔊 **Звук** | Звуковой сигнал при обнаружении нарушения |

## 🚀 Установка

1. Установи [Fabric Loader](https://fabricmc.net/use/installer/) для Minecraft **1.20.1**
2. Скачай [Fabric API 0.92.9+1.20.1](https://modrinth.com/mod/fabric-api)
3. Помести оба `.jar` файла в папку `.minecraft/mods/`
4. Запускай игру!

> При первом запуске автоматически создастся конфиг с базовыми триггерами.

## 🎮 Использование

### Открыть настройки
Нажми **[M]** (русская **[ь]**) находясь в игре (не в чате).

### Как работает обнаружение

```
[22:29:36] [Render thread/INFO]: [System] [CHAT] ʟ | «стᴀжᴇр» 1pirs Обмен? Ко мне!: долбаеб
                                                                        ↑↑↑↑↑             ↑↑↑↑↑↑↑↑
                                                                       НИК           ТРИГГЕР
```

При срабатывании в чате появляется:

```
──────────────────────────────────────
[МОДЕРАЦИЯ] ⚠ Нарушение обнаружено!
Игрок: 1pirs | Причина: Оскорбление игроков. | Рек. мут: 60m
Сообщение: "долбаеб"
Детали: Слово: "долбаеб" | Категория: Оскорбление игроков
Команда: /mute 1pirs 60m 3.10 | Оскорбление игроков.  [ВСТАВИТЬ] [КОПИРОВАТЬ]
──────────────────────────────────────
```

- **[ВСТАВИТЬ]** — открывает чат с готовой командой, жми Enter
- **[КОПИРОВАТЬ]** — копирует команду в буфер обмена

## ⚙️ Настройки в игре

### Категории нарушений

Нажми **[M]** → **⚡ Категории нарушений**

Каждая категория содержит:
- **Название** — отображается в уведомлении
- **Время мута** — `30m`, `1h`, `3h`, `1d`...
- **Правило** — пункт правил сервера (`3.10`, `2.3`...)
- **Причина** — текст причины мута
- **Слова** — список через запятую (регистр не важен)

> 💡 Порядок категорий = приоритет. Первая сработавшая категория применяется.
> **ЛКМ** по категории = редактировать, **ПКМ** = удалить.

### Встроенные категории

| Категория | Мут | Правило | Слов |
|-----------|-----|---------|------|
| Оскорбление игроков | 60m | 3.10 | 26 |
| Аморальное поведение | 60m | 3.3 | 12 |
| Оскорбление родных | 180m | 3.7 | 5 |

## ✈️ Telegram уведомления

### Настройка

1. Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → получи **Bot Token**
2. Получи **Chat ID**:
   - Для личного чата: напиши [@userinfobot](https://t.me/userinfobot)
   - Для канала: добавь бота в канал как администратора, Chat ID начинается с `-100...`
3. В игре: **[M]** → **✈ Telegram: настроить** → введи токен и Chat ID → **Сохранить**
4. Нажми **▶ Отправить тест** чтобы проверить

### Формат сообщения в Telegram

```

━━━━━━━━━━━━━━━━━━━
👤 Игрок: 1pirs
📂 Категория: Оскорбление игроков
🔤 Слово: долбаеб
💬 Сообщение: долбаеб
⏱ Мут: 60m | Правило 3.10
📋 Команда:
/mute 1pirs 60m 3.10 | Оскорбление игроков.
━━━━━━━━━━━━━━━━━━━

```

```json
      "time": "60m",
      "rule": "3.10",
      "reason": "Оскорбление игроков.",
      "enabled": true,
      "words": ["идиот", "тварь", "даун"]
    }
  ]
}
```
